# QuickAccessBoard
This is a 12 key macropad that makes use of a rotary encoder and an OLED display. The body was designed in Fusion, and it uses QMK firmware.

## Features:
- 128x32 OLED Display
- EC11 Rotary Encoder
- 12 Keys

## CAD Model:
The case is secured using 4 M3 bolts and heatset inserts. The PCB is held in place in a sandwich arrangement.

It has two separate printed pieces: the base, and the top cover. 

<img src=assets/CAD.png alt="Schematic"/>

Made in Fusion.

## Firmware Overview
This hackpad uses [QMK](https://qmk.fm/) firmware for everything. 

I will work more on the firmware once the hackpad is physically set up. 

## BOM:
Here should be everything you need to make this hackpad

- 12x Cherry MX Switches
- 12x DSA Keycaps
- 4x M3x5x4 Heatset inserts
- 4x M3x16mm SHCS Bolts
- 12x 1N4148 DO-35 Diodes.
- 1x 0.91" 128x32 OLED Display
- 1x EC11 Rotary Encoder
- 1x XIAO RP2040
- 1x Case (2 printed parts)
